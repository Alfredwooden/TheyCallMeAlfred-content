# TheyCallMeAlfred — Content

Markdown content for the [TheyCallMeAlfred](https://theycallmealfred.com) portfolio site.

## Structure

```
content/
  projects/     # Featured work and case studies
  playground/   # Experiments and prototypes
```

Each file is a single piece of content identified by its filename (e.g. `not-a-design-system.md`).

## Adding Content

1. Create a `.md` file in the appropriate folder (`projects/` or `playground/`)
2. Name the file after the content's `id` field — lowercase, hyphen-separated
3. Follow the frontmatter spec in [STYLEGUIDE.md](./STYLEGUIDE.md)

## Style Guide

See [STYLEGUIDE.md](./STYLEGUIDE.md) for required frontmatter fields, status values, tag conventions, and body formatting rules.

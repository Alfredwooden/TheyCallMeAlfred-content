## Typography Hierarchy

Our typography system uses two font families: Inter for UI text and JetBrains Mono for code. Each text style has been carefully crafted with optimal font sizes, line heights, and weights for perfect readability across all devices.

## Text Styles

Body text styles for general content, from tiny labels to lead paragraphs. These styles maintain consistent vertical rhythm and readability.

## Headline Styles

Headline hierarchy from minor headings to billboard displays. Each level provides clear visual distinction while maintaining harmony in the overall design.

## Font Stack & Implementation

The system leverages CSS custom properties for consistent spacing and sizing. Each text style maps to specific use cases, making it easy for developers to choose the right typography without guesswork.
